前言

最近项目想整一个类似客服系统的东西，网上找了找，整了一个叫 PPMessage 的开源项目叫研究，拿来看一看。今天讲一下使用的情况，首先今天说一下如何搭建这个东西（其实在GitHub上有说明，但是具体上步骤还是有些问题，就来简单讲一讲）。


1、下载项目


在GitHub上你能找到在：

  • Linux/Ubuntu
  • Mac
  • Windows

三个系统里的部署操作，我的是 Mac，Linux/Ubuntu 和 Mac 差不多，Windows 的略微麻烦。本质上差不多，因为这个项目是基于 Python 开发的，所以搭建的工作基本上就是下载项目的各种依赖。

  • Windows 下： Users/xxx\pip\pip.ini
